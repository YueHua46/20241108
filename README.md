# 旋转PDF页面（面试用）

## 项目信息

- 项目名称：旋转PDF页面
- 项目时间：2024-11-08

## 项目说明

- 使用 `react-pdf` 库来实现 PDF 文件的旋转和预览
- 使用 `next.js` 框架来实现页面渲染
- 使用 `tailwindcss` 来实现页面样式
- 使用 `typescript` 来实现类型检查

## 项目部署

- 使用 `vercel` 部署

## 本地运行

- 安装依赖

```bash
npm install
```

- 运行项目

```bash
npm run dev
```

- 访问项目

```bash
http://localhost:3000
```

## 打包项目

```bash
npm run build
```
